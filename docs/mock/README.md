# ViewLoom redesign HTML mock

Static replacement-level mock for all current public page types.

## Included pages
- Portal
- About
- Support
- Twitch: Home, Heatmap, Day Flow, Battle Lines, History, Status
- Kick: Home, Heatmap, Day Flow, Battle Lines, History, Status

## Design direction
- Dense observation surface rather than a SaaS card dashboard
- ViewLoom dark navy palette with restrained Portal/Twitch/Kick accents
- Charts and tables before explanation
- Few cards, few pills, no neon glassmorphism
- Page-specific information architecture
- Responsive layouts and working mock interactions

## Important
The displayed values are mock data. The HTML/CSS structure is intended as a direct visual and structural replacement candidate; real API bindings would replace the static values.

Open `index.html` to review all routes.
