# PR-189 Probe Kick official livestreams endpoint

## Summary

This PR adds a probe script for Kick official livestream listing.

## Why

The candidate registry flow is beta-only. Before changing the collector, we need to verify whether the official livestream listing endpoint returns usable live stream rows with the current token.

## Changes

- Add `workers/collector-kick/scripts/probe-kick-official-livestreams.mjs`.
- Add `pnpm kick:official-livestreams:probe`.

## Scope

Probe script only. No collector runtime changes yet.
