# PR-183 Kick live probe operator scripts

## Summary

This PR makes the Kick live candidate probe easier to run from repo root and adds a short operator runbook.

## What changed

- Added package scripts:
  - `kick:seed-import:check`
  - `kick:seed-import:generate`
  - `kick:live-probe`
  - `kick:live-probe:fast`
- Added `docs/kick-live-probe-operator-runbook.md`.

## Why

PR-182 added the probe script. This PR reduces the operational friction so the operator can run the live probe without remembering the full script path.

## Scope

This PR does not:

- execute D1 writes;
- change collector runtime;
- add discovery;
- claim Kick parity;
- claim directory coverage.
