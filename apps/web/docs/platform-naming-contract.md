# ViewLoom platform naming contract

ViewLoom is the product name. Twitch and Kick identify the observed data source; they are not part of the ViewLoom brand name.

## Global navigation

Provider links use:

- `Twitch data`
- `Kick data`

Do not use provider names alone as ViewLoom navigation labels.

## Feature page hero labels

Battle Lines:

- `/twitch/battle-lines/` uses `TWITCH DATA · RIVALRY`
- `/kick/battle-lines/` uses `KICK DATA · RIVALRY`
- both pages use `Battle Lines` as the H1

Data Status:

- `/twitch/status/` uses `TWITCH DATA · STATUS`
- `/kick/status/` uses `KICK DATA · STATUS`
- both pages use `Data Status` as the H1

Provider names belong in the eyebrow, breadcrumb, description, and metadata, not as a prefix to the feature H1.

## SEO titles

Use the following public titles:

- `Battle Lines for Twitch live streams | ViewLoom`
- `Battle Lines for Kick live streams | ViewLoom`
- `Data Status for Twitch live streams | ViewLoom`
- `Data Status for Kick live streams | ViewLoom`

The document title, Open Graph title, and Twitter title must agree.

## Independence

The public About page must state that ViewLoom is independent and unofficial and is not affiliated with or endorsed by Twitch or Kick.

## Forbidden regressions

- `TWITCH DATA · RIVALRY RADAR`
- `KICK DATA · RIVALRY RADAR`
- `Twitch data status` as the Status H1
- `Kick data status` as the Status H1
- `Twitch Battle Lines — ViewLoom`
- `Kick Battle Lines — ViewLoom`

This contract changes copy and metadata only. It does not change collectors, databases, cron schedules, retention, bindings, or Cloudflare deployment configuration.
