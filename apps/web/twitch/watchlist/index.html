    Local scope and limits

    Browser-only utility
    Local storage

    Browser-specific

    Saved ids stay in this browser’s provider-specific localStorage key. There is no account, cloud sync, cookie fallback, or shared Watchlist URL.

    Evidence

    Bounded observations

    Latest evidence comes from the existing Twitch Heatmap payload. Retained evidence comes from the selected Twitch History payload. Not appearing in either bounded result does not prove offline status, and no complete history is implied.

    Privacy

    No saved ids in metadata

    Saved ids are not placed in the canonical URL, share metadata, or analytics payloads. External Twitch links follow normal browser navigation.
