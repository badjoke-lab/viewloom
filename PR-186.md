# PR-186 Python fallback Kick live probe

## Summary

This PR adds a Python fallback live probe for Kick candidates.

## Why

Local verification showed the Node fetch probe can return very high error counts, while the earlier Python urllib probe was able to find live candidates more reliably.

## What changed

- Added `workers/collector-kick/scripts/probe-kick-live-candidates-python.py`.
- Added package scripts:
  - `kick:live-probe:py`
  - `kick:live-probe:py-safe`

## Scope

The Python script only generates reviewable SQL.

This PR does not:

- execute D1 writes;
- change collector runtime;
- add discovery;
- claim Kick parity;
- claim directory coverage.
